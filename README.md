# D2R AIO — Diablo II: Resurrected Leveling & Crafting Helper

A single-file, offline-capable web tool for Diablo II: Resurrected. Open `index.html` in any modern browser.

## Features

- **Loot Check** — search ~590 uniques, set items, and runewords and see their full stats with min/max roll ranges.
- **Crafting** — a rune tracker (all 33 runes), a "craftable right now" panel, a base-item checklist with the recommended lowest-requirement base for each runeword, and every runeword as a card (class icons, stats, recommended base with str/dex, farm notes). Filter by class, search by name or rune. Endgame runewords start unchecked so they don't clutter your list.
- **Leveling** — per-class fastest-leveling notes: build path, skill order, stat/mana split, gear priority, and mercenary setup.
- **Zones** — per-act zone tables with Normal/Nightmare/Hell area levels, ★85 flags, quest info, and a live "enter your level → XP penalty per zone" calculator.
- **Level 85 Areas** and **Top Farm Runs** quick references.

Tabbed interface; all selections and counts are saved in the browser (localStorage).
